### Missing Skills to Acquire

#### 1. Docker

* Description: Docker is a containerization platform that allows developers to package, ship, and run applications in containers.
* Recommended Course/Training:
	+ Docker Certified Associate (DCA) study guide
	+ Docker Essentials course on Udemy or Coursera
	+ Hands-on experience with Docker by creating and managing containers

#### 2. AWS

* Description: Amazon Web Services (AWS) is a comprehensive cloud computing platform that provides a wide range of services for computing, storage, databases, analytics, machine learning, and more.
* Recommended Course/Training:
	+ AWS Certified Cloud Practitioner - Essentials course on Udemy or Coursera
	+ AWS Developer Essentials course on Udemy or Coursera
	+ Hands-on experience with AWS by creating and managing resources

#### 3. CI/CD (Continuous Integration and Continuous Deployment)

* Description: CI/CD is a software development practice that aims to automate the build, test, and deployment of software applications.
* Recommended Course/Training:
	+ Jenkins or GitLab CI/CD course on Udemy or Coursera
	+ Docker and Kubernetes for CI/CD course on Udemy or Coursera
	+ Hands-on experience with CI/CD tools by creating pipelines

#### 4. Redis

* Description: Redis is an in-memory data store that provides high performance, scalability, and reliability.
* Recommended Course/Training:
	+ Redis Essentials course on Udemy or Coursera
	+ Redis for Developers course on Pluralsight
	+ Hands-on experience with Redis by creating and managing databases

#### 5. Kafka

* Description: Apache Kafka is a distributed streaming platform that provides high-throughput, fault-tolerant, and scalable data processing.
* Recommended Course/Training:
	+ Kafka Essentials course on Udemy or Coursera
	+ Kafka for Developers course on Pluralsight
	+ Hands-on experience with Kafka by creating and managing topics

#### 6. Linux

* Description: Linux is a widely used operating system that provides stability, security, and flexibility.
* Recommended Course/Training:
	+ Linux Essentials course on Udemy or Coursera
	+ Linux for Developers course on Pluralsight
	+ Hands-on experience with Linux by creating and managing systems

### Recommended Certifications

#### 1. Docker Certified Associate (DCA)

* Description: The DCA certification validates an individual's skills in using Docker to develop, ship, and run applications.
* Eligibility: None
* Exam Fee: $100
* Study Materials: Docker Certified Associate study guide

#### 2. AWS Certified Cloud Practitioner - Essentials

* Description: The AWS Certified Cloud Practitioner - Essentials certification validates an individual's understanding of cloud computing concepts, services, and security.
* Eligibility: None
* Exam Fee: $150
* Study Materials: AWS Certified Cloud Practitioner - Essentials study guide

#### 3. Jenkins or GitLab CI/CD Certification

* Description: The Jenkins or GitLab CI/CD certification validates an individual's skills in using continuous integration and continuous deployment tools.
* Eligibility: None
* Exam Fee: $100-$200
* Study Materials: Jenkins or GitLab CI/CD study guide

#### 4. Redis Essentials Certification

* Description: The Redis Essentials certification validates an individual's skills in using Redis to develop, deploy, and manage applications.
* Eligibility: None
* Exam Fee: $50
* Study Materials: Redis Essentials study guide

#### 5. Kafka Essentials Certification

* Description: The Kafka Essentials certification validates an individual's skills in using Apache Kafka to develop, deploy, and manage streaming data pipelines.
* Eligibility: None
* Exam Fee: $100
* Study Materials: Kafka Essentials study guide
